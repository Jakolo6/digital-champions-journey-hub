# AI Leadership Learning Platform

A comprehensive learning platform designed to empower leaders with AI capabilities and drive organizational transformation through artificial intelligence.

## Overview

This platform provides leaders with the knowledge, tools, and community support needed to successfully navigate AI transformation. Through structured learning modules, practical applications, and collaborative networking, leaders develop the expertise to drive innovation and lead confidently in the AI era.

## Key Features

- **AI-Powered Learning**: Comprehensive training modules covering AI fundamentals to advanced applications
- **Leadership Development**: Strategic frameworks for leading AI transformation initiatives
- **Practical Innovation**: Real-world case studies and hands-on exercises
- **Collaborative Network**: Connect with fellow leaders navigating AI transformation
- **Progress Tracking**: Monitor learning progress and earn credits
- **Interactive Content**: Engaging video content and interactive learning materials

## Technologies

This project is built with modern web technologies:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI component library
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - High-quality React components
- **React Router** - Client-side routing

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn package manager

### Installation

```sh
# Clone the repository
git clone https://github.com/Jakolo6/Learning_Platform.git

# Navigate to the project directory
cd Learning_Platform

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```sh
npm run build
```
